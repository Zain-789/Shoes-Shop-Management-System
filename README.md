# Shoes Shop Management System

A complete professional Windows Desktop Application for managing shoes shop operations including inventory, sales, purchases, customers, and reporting.

## 🎯 Features

### ✅ Core Modules
- **Login System** - Secure owner/admin authentication
- **Dashboard** - Real-time analytics and quick actions
- **Product Management** - Full CRUD operations with variants
- **Inventory Management** - Automatic stock tracking
- **Purchase Module** - Supplier invoice management
- **Sales Module** - Complete POS with bill generation
- **Customer Management** - Customer profiles and history
- **Supplier Management** - Supplier information tracking
- **Expense Tracking** - Shop expense management
- **Reports** - Daily, Monthly, and Stock reports
- **Backup/Restore** - Database backup system

### 🎨 User Interface
- Professional WPF Modern Design
- Left Navigation Menu
- Responsive Dashboard
- Data Tables with Search
- Form Validations
- Real-time Stock Updates

### 💾 Database
- SQLite Database
- Proper Relational Schema
- Foreign Keys & Constraints
- Automatic Migrations

## 🛠️ Technology Stack

- **Framework:** C# .NET Framework / .NET Core
- **UI:** WPF (Windows Presentation Foundation)
- **Database:** SQLite
- **Architecture:** MVVM Pattern
- **Tools:** Visual Studio 2019+

## 📋 Requirements

- Windows 7 or Higher
- .NET Framework 4.7.2+ or .NET 6+
- SQLite (included)

## 🚀 Installation

1. Extract the project files
2. Open `ShoesShopMS.sln` in Visual Studio
3. Build the solution (Build > Build Solution)
4. Run the application
5. Default Login:
   - Username: `admin`
   - Password: `admin`
